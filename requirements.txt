# Notes
# ============================================================================
# 1. masktheface is included as a local directory in this project
#    It requires the above dependencies to function properly
#
# 2. For dlib on Windows, you may need to install from wheel file:
#    pip install dlib-19.22.99-cp310-cp310-win_amd64.whl
#    (wheel file should be in project directory)
#
# 3. For GPU support, install CUDA-enabled PyTorch:
#    Visit: https://pytorch.org/get-started/locally/
#    Select your CUDA version and follow the installation instructions
#
# 4. InsightFace models will be auto-downloaded on first use
#    Models are stored in: ~/.insightface/models/
#    Model: buffalo_l (default, ~500MB)
#
# 5. VGGFace2 model will be auto-downloaded on first use
#    Model is cached by facenet-pytorch
#    Model: InceptionResnetV1 pretrained='vggface2' (~100MB)
#
# 6. dlib shape predictor should be in dlib_models/ directory
#    File: shape_predictor_68_face_landmarks.dat (~100MB)
#    Download from: http://dlib.net/files/shape_predictor_68_face_landmarks.dat.bz2
#
# 7. Installation order (if encountering issues):
#    a) pip install torch torchvision (CPU or CUDA version)
#    b) pip install opencv-python opencv-contrib-python
#    c) pip install insightface facenet-pytorch
#    d) pip install dlib (or use wheel file on Windows)
#    e) pip install -r requirements.txt (for remaining packages)
